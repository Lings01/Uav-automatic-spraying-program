操作:
	在无人机正常起飞后，打开遥控开始跑程序，接收到视觉稳定识别150ms后开始喷洒，一但接收到不是薇甘菊
	则立刻停止喷洒，防止识别不稳时频繁启停电机

失控保护：
	关闭遥控器，立刻停止喷洒药剂


打开程序：
	1.需安装Keil uVision5，可以打开整个工程
		1.）双击Project文件
		2.）双击project工程打开即可

	2.用记事本打开单个文件，如头文件（.h）和文件（.c）
		1.）usart2.h  遥控器头文件
		    usart2.c  遥控器收发数据
		2.）usart5.h  视觉通信头文件
		    usart5.c  视觉通信文件，包含收发
		3.）control.h 控制头文件
		    control.c 控制文件，喷洒过程的逻辑
